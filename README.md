# Customer Churn & Revenue Risk Analysis

This project analyzes customer churn and recurring revenue risk for a B2C subscription-based business.

## Objective
The goal of this analysis is to understand recent churn trends, identify high-risk customer segments, and estimate the amount of recurring revenue at risk due to churn.

## Tools
- SQL
- Python (pandas, numpy)
- Power BI

Project setup complete. Data understanding and cleaning in progress.
